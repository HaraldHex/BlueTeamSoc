# SOC Threat Intelligence Enrichment Tool 

A Python-based SOC simulation tool that enriches log data with threat intelligence classification.

## Features
- Extracts IP addresses from logs
- Matches IPs against threat intelligence database
- Classifies IPs (safe, suspicious, malicious)
- Generates SOC-style security report

## Concepts used
- Log parsing
- Regex
- Threat intelligence simulation
- Risk classification logic
- SOC workflow simulation
